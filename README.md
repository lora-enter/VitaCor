# VitaCor – забота о сердце

VitaCor – это десктопное приложение, созданное для людей, перенёсших инфаркт или заботящихся о здоровье сердца. Приложение помогает соблюдать диету, получать ежедневные рецепты, общаться с чат-ботом Vita и заказывать персональное питание у профессиональных шеф-поваров.

Интерфейс выполнен в больничной красно-белой гамме с плавными анимациями и адаптивным дизайном.

---

## 📋 Основные возможности

- **Регистрация по номеру телефона** с имитацией отправки SMS-кода (тестовый аккаунт: `+79991234567`, код `123456`).
- **Настройка диеты и аллергий** – пользователь указывает тип диеты (например, низкосолевая) и запрещённые продукты.
- **Чат с ботом Vita**:
  - Ежедневный персонализированный рецепт в 8:00 утра.
  - Кнопка «Новый рецепт» – мгновенная замена блюда.
- **Каталог шеф-поваров**:
  - Прокрутка списка колёсиком мыши.
  - Подробная карточка повара с рейтингом, ценами и биографией.
  - Кнопка «Заказать» с имитацией оформления заказа.
- **Настройки профиля** (имя, фамилия, телефон) в правом верхнем углу.
- **Адаптивный интерфейс** – окно можно расширить, все элементы перестраиваются.
- **Анимированный фон** – плавающие полупрозрачные медицинские кресты.

---
Возможные улучшения

- Интеграция с реальным API для отправки SMS.
- Сохранение данных пользователя.
- Расширенная система рецептов с фильтрацией по КБЖУ (калориям, белкам, жирам, углеводам).
- Уведомления о приёме лекарств.
